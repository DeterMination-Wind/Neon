```powershell
gh release edit v9.0.0 --notes "$(cat <<'NOTES_EOF'
# Neon v9.0.0 — 19-in-1

## 中文

Neon 升级为 **19 合 1**，新并入四个模块：

- **ForeignServerTranslator**（外语服务器翻译）：标记外语服务器，自动翻译收发聊天与服务器文本，支持 Microsoft Translator 与 OpenAI 兼容接口。
- **Tripwire**（地理围栏报警）：在世界中创建地理围栏，检测指定单位穿越并发送 Toast / 聊天 / marker 提醒。
- **BetterPolyAi**（Poly 建造辅助）：从 MindustryX 提取的 Poly 建造辅助，可按快捷键启停，仅执行玩家自己的建造/拆除规划。
- **AdvancedReplace**（地图编辑高级替换）：为地图编辑器补充同色染色墙填充、染色地板填充、染色墙画笔等工具模式，支持 CIEDE2000 色差宽容度设置。

其他改动：

- 四个新模块均接入 `bekBundled / bekBuildSettings`，设置项统一挂入 Neon 折叠分组。
- 除 Mod Updater 外，bundled 子模组不再执行各自独立的 GitHub 更新检查。
- Neon 设置页改为各模块独立折叠分组，打开设置分类时默认全部折叠。
- 移除 Neon 独立"更新"设置分组，`WhoUsesThisBuilding` 触发键迁移到原版控制设置。
- `ServerPlayerDataBase` 新增语义搜索（基于本地嵌入模型的向量检索）。
- `PatchViewer` 同步到本地独立仓库 HEAD，补上异常 datapatch 快照保护。
- Profiler 增加设置页一键启停查看按钮，补齐 Mod Updater 内置 repo 映射。

## English

Neon is now a **19-in-1** package with four newly bundled modules:

- **ForeignServerTranslator**: Marks foreign-language servers and translates incoming, outgoing, and server chat text. Supports Microsoft Translator and OpenAI-compatible endpoints.
- **Tripwire**: Creates geofence lines in-world and alerts when selected unit types cross them. Supports toast, chat, and marker alerts.
- **BetterPolyAi**: Poly build assist extracted from MindustryX, toggleable by keybind. Executes only your own build/deconstruct plans.
- **AdvancedReplace**: Adds same-color colored-wall fill, color-aware colored-floor fill, and colored-wall brush modes to the map editor with CIEDE2000 tolerance settings.

Other changes:

- All four modules use `bekBundled / bekBuildSettings` hooks and mount into Neon grouped settings.
- Except for Mod Updater, bundled submodules no longer run their own GitHub update checks.
- Neon settings page now uses independently collapsible module groups, collapsed by default.
- Removed Neon's standalone Update settings group; moved `WhoUsesThisBuilding` trigger to vanilla Controls.
- `ServerPlayerDataBase` now supports semantic search via local embedding model vector retrieval.
- Synced `PatchViewer` with the local standalone repository HEAD and added malformed datapatch snapshot protection.
- Added a one-click profiler toggle/view button and filled Mod Updater built-in repo mappings.
NOTES_EOF
)"
```
