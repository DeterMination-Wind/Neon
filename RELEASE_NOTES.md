# Neon v5.2.3 Release Notes

## English

- Fixed a bundled SPDB overlay focus regression where lightweight query controls could become non-responsive during active pointer interactions.
- Fixed bundled SPDB auto-trace sending admin actions to admin targets, eliminating repeated server warnings (`attempted to perform admin action on nonexistant or admin player.`).
- Synced the latest local SPDB snapshot with the above fixes into Neon.

## 中文

- 修复整合版 SPDB 覆盖层焦点回归：指针交互时轻量查询按钮偶发无响应。
- 修复整合版 SPDB 自动 Trace 对管理员目标发送管理操作，避免服务端后台重复出现 `attempted to perform admin action on nonexistant or admin player.` 警告。
- 将上述修复同步到 Neon 当前整合快照。
